# Vaibhav Suneja — Portfolio Website 🎨

This is the personal portfolio of **Vaibhav Suneja**, a 3D Artist & Motion Graphics Designer.

## Instructions 🛠️

The project uses GSAP Club plugins (trial version). To host the site, you'll need the paid Club plugins.
Check here: https://gsap.com/docs/v3/Installation/

**Techstack** — React, TypeScript, GSAP, ThreeJS, WebGL, HTML, CSS, JavaScript
